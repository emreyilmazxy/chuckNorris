# Chuck Norris Şaka Uygulaması

JavaScript pratik projesi: Chuck Norris API'sinden rastgele şakalar çeken basit bir web uygulaması.

## Özellikler
- Tek tıkla yeni bir Chuck Norris şakası getirir.
- Yükleniyor ve hata durumları kullanıcıya ifade edilir.
- Responsive kart tasarımı ile modern bir görünüm sağlar.

## Kurulum
1. Bu klasörü yerel makinenize indirin ya da kopyalayın.
2. `index.html` dosyasını modern bir tarayıcıyla açın.

## Proje Yapısı
```
index.html
css/
  styles.css
img/
  Ekran görüntüsü 2025-11-14 060506.png
js/
  app.js
```

## Kullanım
1. Sayfayı açın.
2. `new joke` butonuna her tıklayışınızda yeni bir şaka görüntülenir.

- Şakalar `https://api.chucknorris.io/jokes/random` uç noktasından çekilir.
- API yanıtı alınırken basit hata yakalama uygulanmıştır.
